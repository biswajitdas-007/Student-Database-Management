# How to Run Project

Clone the app

```
git clone <url>
```

How to run Backend:

```
cd Backend
npm install
npm start
```

How to run Frontend:

```
cd Frontend
npm install
npm start
```

To login as admin:

```
add .masaischool.com at last in your email
example: abc@gmail.masaischool.com

```

To login as student:

```
enter the gmail present in the database

email is given at the time of student details add into database

```
